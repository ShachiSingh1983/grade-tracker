# Grade Tracker

A web app to track CGPA and predict future grades based on upcoming courses.

## Features
- Add courses with credits and grades
- Auto-calculates CGPA using standard 10-point scale
- Predicts your CGPA based on future course scenarios
- Data persists across sessions using localStorage

HTML · CSS · JavaScript
